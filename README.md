# TorturedSoulsTTS

TorturedSoulsTTS is a World of Warcraft addon that plays text-to-speech alerts when Vessel of Tortured Souls orbs spawn or are picked up.

## Features

- Separate TTS alerts for orb spawn and orb pickup events
- Per-alert enable or disable checkboxes
- Custom text for each alert
- Adjustable TTS rate and volume for each alert
- In-game configuration panel opened with `/vtstts` or `/vts`

## Installation

1. Download or clone this repository.
2. Place the `TorturedSoulsTTS` folder in your WoW AddOns directory:
   `World of Warcraft/_retail_/Interface/AddOns/`
3. Reload the game or restart WoW.

## Usage

- Use `/vtstts` or `/vts` to open the configuration panel.
- Choose the TTS voice.
- Configure spawn and pickup alerts independently.
- Use the test buttons to preview each alert.
